# CHZZK-PureView

치지직 1080p 환경으로 보기 위한 Chrome/Edge 확장 프로그램입니다.

## 기능

- 1080p 시청 환경 유지
- 광고, P2P그리드 우회

## 설치 방법

1. Chrome 또는 Edge에서 `chrome://extensions` 또는 `edge://extensions`를 엽니다.
2. 우측 상단(Chrome)/좌측 하단(Edge)의 `개발자 모드`를 켭니다.
3. `압축해제된 확장 프로그램을 로드` 또는 `압축 풀린 파일 로드`를 누릅니다.
4. 압축을 푼 폴더 또는 `CHZZK-PureView_unpacked` 폴더를 선택합니다.
5. 치지직 페이지를 새로고침합니다.
